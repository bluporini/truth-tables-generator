# Truth Table Generator


Application to generate **truth tables** from logical propositions. The interface is a console, in Spanish.  


## Supported Operators


| Operator | Symbol | 
|----------|--------|
| NOT | `~` |
| AND | `∧` |
| OR | `∨` |
| IMPLIES | `->` |
| BICONDITIONAL | `<->` |



## How to Run

### Executable Archive

To run the application without Python, use the pre-built executable: ```dist/main.exe```

Double-click the file to open the application. No Python installation is required.

### Using Python

Run the script:

```bash
python main.py
